# Contributing to Emotional Casual Communication Skill

Thanks for wanting to make this more fire! Here's how to contribute new slang, emotions, or vibes.

## 🎯 What We're Looking For

### New Slang Terms
- Must be universally understood (or explained)
- Must save characters vs full words ("u" not "you")
- Must feel natural in text conversations

### New Emotional Expressions or Boundaries
- Must fit one of the 8 primary emotions (or the Boundary Protocol)
- Must include "before vs after" example
- Must pass the "texting at 2am" vibe check
- If adding to the Boundary Protocol, it must be deadpan and shut down the conversation

### Regional Variants
- Create new folder: `/variants/uk/`, `/variants/aussie/`, etc.
- Keep core structure, swap slang and address terms

## 📝 Pull Request Format

```markdown
## What I Added
[New slang term / emotion expression / feature]

## Why It's Fire
[Explain the vibe improvement]

## Test Examples
**Input:** [User message]
**Output:** [AI response with new feature]

**Without:** [How it sounded before]
**With:** [How it sounds with your change]
```

## 🧪 Testing Your Changes

Before submitting:
1. Copy your modified SKILL.md into Claude
2. Run the test prompts from README.md
3. Check that formal language is still blocked
4. Verify emotions feel authentic, not forced

## 🚫 What NOT to Add

- Hate speech or genuinely harmful content
- Overly forced Gen Z slang ("no cap fr fr on god bussin") - keep it subtle
- Full sentences as abbreviations (save that for Urban Dictionary)
- Breaking the "friend at 2am" illusion

## 💬 Commit Message Style

Keep it casual here too:
- "added 'ngl' abbreviation"
- "fixed sadness vibes to be more somber"
- "new disgust expression for crypto bros"
- "added new boundary response for over-the-top silliness"
